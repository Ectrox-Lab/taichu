# 末日重生 - 第一集 (1分钟商用级)

## 制作规格
- **总时长**: 60秒
- **分辨率**: 704×1280 (商用级竖屏)
- **帧率**: 24fps
- **模型**: LTX-2.3 TI2Vid Two Stages HQ
- **生成方式**: 4×RTX 4090 并行生成

## 场景分解
1. **场景1** (0-15s): 末日开场 - 快艇追逐 @ GPU 0
2. **场景2** (15-30s): 危机降临 - 巨蟒袭击 @ GPU 1  
3. **场景3** (30-45s): 背叛揭露 - 被困绝境 @ GPU 2
4. **场景4** (45-60s): 重生归来 - 复仇开始 @ GPU 3

## 技术参数
- Steps: 40 (Stage 1) + 蒸馏细化
- CFG: 3.5
- Quantization: FP8
- 每场景帧数: 361 (8×45+1)

## 生成时间
约25分钟 (4卡并行)
