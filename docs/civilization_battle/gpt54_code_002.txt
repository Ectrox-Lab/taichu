## 1. 系统架构

查询优化器接收SQL查询，经过解析、逻辑计划生成、基于代价的物理计划搜索，最终输出执行计划。

- 统计信息管理器（直方图）
- 代价模型（IO + CPU + Memory）

### 选择率估计

- **范围条件**：`A.y > 100`
  - 使用直方图定位桶
  - 桶内比例 = (high - 100) / (high - low) × freq
  
- **等值条件**：`A.x = B.x`
  - 选择率 = 1 / max(ndistinct(A.x), ndistinct(B.x))

- **多条件**：假设独立性，选择率相乘

- **GROUP BY**：用分组列 NDV 乘积并截断到输入行数

## 3. 代价模型

### 基本代价因子

```python
class CostModel:
    def __init__(self):
        self.seq_page_cost = 1.0
        self.random_page_cost = 4.0
        self.cpu_tuple_cost = 0.01
        self.cpu_index_tuple_cost = 0.005
        self.memory_cost_per_byte = 0.00001
```

### 各操作代价

**SeqScan**：
```
Cost = pages × seq_page_cost + rows × cpu_tuple_cost
```

**IndexScan**：
```
Cost = index_pages × random_page_cost + rows × (random_page_cost + cpu_tuple_cost)
```

**NestedLoopJoin**：
```
Cost = N_outer × cost_inner + N_outer × N_inner × cpu_operator_cost
```

**HashJoin**（假设内存足够）：
```
Cost = cost_left + cost_right + (|left| + |right|) × cpu_operator_cost
```

如果 inner_rows × row_size > Memory，需 spill to disk（增加 IO 成本）。

**Sort**：
```
Cost = 2 × data_size × seq_page_cost + rows × log(rows) × cpu_operator_cost
```

**Aggregate**：
```
Cost = rows × cpu_operator_cost (+ hash table memory cost)
```

## 4. 计划枚举（动态规划）

```python
class Optimizer:
    def optimize(self, query):
        tables = list(query.tables)
        best = {}  # subset -> PlanNode
        
        # 初始化单表
        for table in tables:
            plan = self.create_scan_plan(table, query.conditions.get(table))
            best[frozenset([table])] = plan
        
        # 动态规划枚举连接
        for size in range(2, len(tables) + 1):
            for subset in combinations(tables, size):
                key = frozenset(subset)
                best_plan = None
                
                # 枚举所有分割
                for left in proper_subsets(subset):
                    right = subset - left
                    left_plan = best[frozenset(left)]
                    right_plan = best[frozenset(right)]
                    
                    # 尝试不同连接方法
                    for method in ['nested_loop', 'hash', 'merge']:
                        if self.can_apply(method, left_plan, right_plan):
                            cost = self.cost_model.join_cost(method, left_plan, right_plan)
                            if best_plan is None or cost < best_plan.cost:
                                best_plan = JoinNode(left_plan, right_plan, method, cost)
                
                best[key] = best_plan
        
        # 添加聚合
        final_plan = self.add_aggregation(best[frozenset(tables)], query.group_by)
        return final_plan
```

### DP 关键优化

- **Left-Deep Tree**：只考虑左深树，减少搜索空间
- **Pruning**：若计划 A 代价高于 B 且输出行数更多，丢弃 A
- **Top-K**：每个子集保留最优的 K 个计划

## 5. 物化策略

**决策依据**：
- 中间结果需要多次使用 → 物化
- 下游操作需要随机访问 → 物化
- 中间结果很小 → 可物化
- 内存足够 → 流水线

**HashJoin**：必须物化内表构建哈希表
**NestedLoopJoin**：可流水线驱动外表
**Sort**：必须物化（外排序）
**Aggregate**：HashAgg 物化哈希表，SortAgg 流水线

```python
def should_materialize(self, plan, parent_requirements):
    if plan.output_rows < MEMORY_THRESHOLD:
        return False  # 小结果，可直接传递
    if parent_requirements.random_access:
        return True  # 需要随机访问，物化
    if plan.operator in [SORT, HASH_AGG]:
        return True  # 操作本身需要物化
    return False  # 默认流水线
```

## 7. 核心实现（Python）

```python
from dataclasses import dataclass
from typing import List, Dict, Optional, Set, FrozenSet
from enum import Enum
import math

class OpType(Enum):
    SEQ_SCAN = "Seq Scan"
    INDEX_SCAN = "Index Scan"
    NESTED_LOOP = "Nested Loop"
    HASH_JOIN = "Hash Join"
    MERGE_JOIN = "Merge Join"
    SORT = "Sort"
    AGGREGATE = "Aggregate"

@dataclass
class PlanNode:
    op_type: OpType
    cost: float
    rows: int
    children: List['PlanNode']
    table: Optional[str] = None
    condition: Optional[str] = None
    
    def explain(self, indent=0):
        prefix = "  " * indent
        result = f"{prefix}{self.op_type.value}  (cost={self.cost:.2f} rows={self.rows})"
        if self.table:
            result += f" on {self.table}"
        if self.condition:
            result += f" where {self.condition}"
        for child in self.children:
            result += "\n" + child.explain(indent + 1)
        return result

class CostModel:
    def __init__(self):
        self.seq_page_cost = 1.0
        self.random_page_cost = 4.0
        self.cpu_tuple_cost = 0.01
        self.memory_cost_per_row = 0.001
    
    def seq_scan_cost(self, pages: int, rows: int) -> float:
        return pages * self.seq_page_cost + rows * self.cpu_tuple_cost
    
    def index_scan_cost(self, index_pages: int, rows: int) -> float:
        return (index_pages * self.random_page_cost + 
                rows * (self.random_page_cost + self.cpu_tuple_cost))
    
    def hash_join_cost(self, left_rows: int, right_rows: int, 
                       left_size: int, right_size: int) -> float:
        build_cost = right_rows * self.cpu_tuple_cost
        probe_cost = left_rows * self.cpu_tuple_cost
        # Memory check
        memory_cost = 0
        if right_size > 10 * 1024 * 1024:  # 10MB threshold
            memory_cost = right_size * self.memory_cost_per_row * 2  # spill
        return build_cost + probe_cost + memory_cost
    
    def nested_loop_cost(self, outer_rows: int, inner_cost: float,
                         inner_rows: int) -> float:
        return outer_rows * inner_cost + outer_rows * inner_rows * 0.005
    
    def sort_cost(self, rows: int, width: int) -> float:
        pages = rows * width / 8192
        return 2 * pages * self.seq_page_cost + rows * math.log2(max(rows, 2)) * 0.01

class Statistics:
    def __init__(self):
        self.tables = {
            'A': (100000, 1000, {'x': 1000, 'y': 200, 'z': 50}),
            'B': (50000, 500, {'x': 1000, 'w': 100}),
            'C': (10000, 100, {'x': 1000})
        }
    
    def get_table_stats(self, table: str) -> Tuple[int, int]:
        return self.tables[table][0], self.tables[table][1]
    
    def get_column_ndv(self, table: str, column: str) -> int:
        return self.tables[table][2].get(column, 100)
    
    def estimate_selectivity(self, table: str, condition: str) -> float:
        # 简化：范围条件 0.3，等值条件 1/NDV
        if '>' in condition or '<' in condition:
            return 0.3
        elif '=' in condition:
            col = condition.split('=')[0].strip()
            ndv = self.get_column_ndv(table, col)
            return 1.0 / ndv
        return 0.5

class Optimizer:
    def __init__(self):
        self.cost_model = CostModel()
        self.stats = Statistics()
    
    def create_scan_plan(self, table: str, condition: Optional[str]) -> PlanNode:
        rows, pages = self.stats.get_table_stats(table)
        if condition:
            selectivity = self.stats.estimate_selectivity(table, condition)
            rows = int(rows * selectivity)
        
        # 选择性高时用索引扫描
        if condition and rows < 10000:
            cost = self.cost_model.index_scan_cost(pages // 10, rows)
            return PlanNode(OpType.INDEX_SCAN, cost, rows, [], table, condition)
        else:
            cost = self.cost_model.seq_scan_cost(pages, rows)
            return PlanNode(OpType.SEQ_SCAN, cost, rows, [], table, condition)
    
    def estimate_join_rows(self, left_rows: int, right_rows: int,
                          left_table: str, right_table: str,
                          join_col: str) -> int:
        ndv_left = self.stats.get_column_ndv(left_table, join_col)
        ndv_right = self.stats.get_column_ndv(right_table, join_col)
        return int(left_rows * right_rows / max(ndv_left, ndv_right))
    
    def create_join_plan(self, left: PlanNode, right: PlanNode,
                        join_type: str, left_table: str, right_table: str,
                        join_col: str) -> PlanNode:
        rows = self.estimate_join_rows(left.rows, right.rows,
                                       left_table, right_table, join_col)
        
        if join_type == 'hash':
            cost = self.cost_model.hash_join_cost(left.rows, right.rows, 100, 100)
            total_cost = left.cost + right.cost + cost
            return PlanNode(OpType.HASH_JOIN, total_cost, rows, [left, right])
        elif join_type == 'nested_loop':
            cost = self.cost_model.nested_loop_cost(left.rows, right.cost, right.rows)
            total_cost = left.cost + cost
            return PlanNode(OpType.NESTED_LOOP, total_cost, rows, [left, right])
        else:
            # merge join needs sort
            sort_cost = self.cost_model.sort_cost(left.rows, 100)
            total_cost = left.cost + right.cost + sort_cost + rows * 0.01
            return PlanNode(OpType.MERGE_JOIN, total_cost, rows, [left, right])
    
    def optimize(self, tables: List[str], conditions: Dict[str, str],
                 join_conditions: List[Tuple[str, str, str]]) -> PlanNode:
        # DP 表：子集 -> 最优计划
        best: Dict[FrozenSet[str], PlanNode] = {}
        table_plans: Dict[str, PlanNode] = {}
        
        # 初始化单表
        for table in tables:
            plan = self.create_scan_plan(table, conditions.get(table))
            best[frozenset([table])] = plan
            table_plans[table] = plan
        
        # 动态规划枚举连接
        from itertools import combinations
        for size in range(2, len(tables) + 1):
            for subset_tuples in combinations(tables, size):
                subset = frozenset(subset_tuples)
                best_plan = None
                
                # 枚举所有分割
                for left_table in subset_tuples:
                    right_tables = subset - {left_table}
                    if not right_tables:
                        continue
                    
                    left_plan = best[frozenset([left_table])]
                    right_plan = best[right_tables]
                    
                    # 尝试不同连接方法
                    for method in ['hash', 'nested_loop']:
                        join_col = 'x'  # 简化：假设都按 x 连接
                        plan = self.create_join_plan(left_plan, right_plan,
                                                    method, left_table,
                                                    list(right_tables)[0],
                                                    join_col)
                        if best_plan is None or plan.cost < best_plan.cost:
                            best_plan = plan
                
                if best_plan:
                    best[subset] = best_plan
        
        final_plan = best[frozenset(tables)]
        
        # 添加聚合（如果有 GROUP BY）
        if final_plan.rows > 0:
            agg_cost = final_plan.cost + final_plan.rows * 0.01
            final_plan = PlanNode(OpType.AGGREGATE, agg_cost, 
                                 min(final_plan.rows, 50), [final_plan])
        
        return final_plan

# 测试
def test_optimizer():
    opt = Optimizer()
    
    # 测试1：两表连接
    plan = opt.optimize(['A', 'B'], {'A': 'y > 100'}, [('A', 'B', 'x')])
    print("Test 1: Two-table join")
    print(plan.explain())
    print()
    
    # 测试2：三表连接
    plan = opt.optimize(['A', 'B', 'C'], {}, [('A', 'B', 'x'), ('B', 'C', 'x')])
    print("Test 2: Three-table join")
    print(plan.explain())

if __name__ == "__main__":
    test_optimizer()
```
