# 国内内网穿透方案（推荐）

## 方案 1: NATAPP（最推荐）

### 官网
https://natapp.cn

### 注册与配置步骤

1. **注册账号**
   - 访问 https://natapp.cn/register
   - 使用手机号注册（国内用户友好）

2. **购买免费隧道**
   - 登录后点击「购买隧道」
   - 选择「免费隧道」
   - 协议：Web
   - 本地端口：9000
   - 隧道名称：feishu-webhook

3. **获取 authtoken**
   - 点击「我的隧道」
   - 复制「authtoken」

4. **下载客户端**
   ```bash
   cd /tmp
   wget https://cdn.natapp.cn/assets/downloads/clients/2_3_9/natapp_linux_amd64_2_3_9.zip
   unzip natapp_linux_amd64_2_3_9.zip
   chmod +x natapp
   sudo mv natapp /usr/local/bin/
   ```

5. **启动 NATAPP**
   ```bash
   natapp -authtoken=YOUR_TOKEN_HERE
   ```

6. **获取公网 URL**
   - 启动后会显示类似：`http://xxxxx.natappfree.cc`
   - 这就是你的 Webhook URL

7. **配置飞书**
   ```
   http://xxxxx.natappfree.cc/webhook/feishu
   ```

## 方案 2: Sunny-Ngrok（国内版）

### 官网
https://www.ngrok.cc

### 特点
- 国内服务器，速度快
- 支持支付宝/微信付款
- 有免费隧道

### 注册步骤
1. 访问 https://www.ngrok.cc/register
2. 注册账号
3. 开通隧道
4. 下载客户端配置

## 快速开始（NATAPP）

```bash
# 1. 下载
wget https://cdn.natapp.cn/assets/downloads/clients/2_3_9/natapp_linux_amd64_2_3_9.zip
unzip natapp_linux_amd64_2_3_9.zip

# 2. 运行（替换 YOUR_TOKEN）
./natapp -authtoken=YOUR_TOKEN

# 3. 看到 URL 后，配置到飞书
# 例如: http://abc123.natappfree.cc/webhook/feishu
```
