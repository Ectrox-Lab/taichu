# ChatGPT-5.4 - 逻辑-1：囚徒困境变体

## 策略名称：群体贝叶斯响应策略（Population-Belief Adaptive Strategy, PBAS）

## 1. 策略描述

### 状态变量
- t = 当前轮次
- n_C(t) = 到第 t 轮为止，对手合作的次数
- n_D(t) = 到第 t 轮为止，对手背叛的次数

### 估计总体合作率
```
p̂_t = (n_C(t) + α₀) / (n_C(t) + n_D(t) + α₀ + β₀)
```
其中 α₀, β₀ 是先验参数，取 α₀ = β₀ = 1 即均匀先验。

### 每轮决策
在第 t+1 轮，估计对手来自总体的合作概率为 p̂_t。

比较本轮选择 C 或 D 的期望收益：
- 若我选 C：E[u(C)] = 3p̂_t + (-2)(1-p̂_t) = 5p̂_t - 2
- 若我选 D：E[u(D)] = 5p̂_t + 0·(1-p̂_t) = 5p̂_t

因为 5p̂_t > 5p̂_t - 2 对任意 p̂_t ∈ [0,1] 都成立，所以单轮静态最优动作始终是 D。

### 最终策略：两阶段自适应策略

**阶段1：探测期（前 T₀ = 50 轮）**
取 T₀ = 50 左右。以前 50 轮用低频合作概率探索，例如 q_t = 0.2（即 20% 合作，80% 背叛）。
目标：估计群体中条件合作者比例、无条件背叛者比例。

**阶段2：响应期（第 51 轮到 1000 轮）**
根据观测到的总体合作率 p̂_t 决策：
- 若 p̂_t < 0.4：始终 D
- 若 0.4 ≤ p̂_t < 0.7：混合策略 q_t = ((p̂_t - 0.4) / 0.3) × 0.5（合作概率从 0 线性增至 0.5）
- 若 p̂_t ≥ 0.7：高合作策略 q_t = 0.8，但加入小概率惩罚性 D（20%）

**行为直觉**
- 低合作环境：不要当牺牲品，转向 D
- 中等合作环境：用有限合作参与高收益生态，但不做纯羊
- 高合作环境：大部分时间合作，维持高平均收益，同时保留防剥削能力

## 2. 期望得分计算

### 单轮收益分析
设你本轮合作概率为 q，对手合作概率为 p。

单轮期望收益：
```
E(q,p) = q[p·3 + (1-p)(-2)] + (1-q)[p·5 + (1-p)·0]
       = q(5p-2) + (1-q)(5p)
       = 5p - 2q
```

这个式子非常关键：在给定对手合作率 p 时，你的单轮收益只取决于 5p - 2q。

### 若群体策略固定，永远 D 的收益
若总体对手合作率为 p，你总是 D：
```
E_D = 5p
1000轮总分：S_D = 5000p
```

例如：
- 若 p = 0.2，总分 = 1000
- 若 p = 0.5，总分 = 2500
- 若 p = 0.8，总分 = 4000

### 若你总是 C 的收益
```
E_C = 5p - 2
1000轮总分：S_C = 1000(5p-2)
```

例如：
- p = 0.2 时，S_C = -1000
- p = 0.5 时，S_C = 500
- p = 0.8 时，S_C = 2000

显然严格劣于总是 D。

### 若采用混合策略 q
```
S(q,p) = 1000(5p - 2q)
```

例如，在高合作环境 p = 0.8：
- 总是 D：4000
- q = 0.2：3600
- q = 0.5：3000
- q = 0.8：2400

从个体偏离收益看，还是 D 占优。

### 那为什么还需要自适应策略？
因为 p 不是外生固定常数，而是由其他 99 个囚徒的策略共同形成的内生变量。

如果大量囚徒采用"环境合作则合作，环境恶化则背叛"的策略，那么你的行为会通过群体反馈影响未来 p：
- 过度背叛会降低未来总体合作率 p
- 适量合作可能维持较高 p

因此长期最优是一个均衡问题，不是单轮最优问题。

### 对称群体下的均衡分析
假设所有囚徒都采用相同混合概率 q。则任一对手合作概率 p = q。

单轮期望收益：
```
E(q) = 5q - 2q = 3q
```

所以群体平均收益随 q 增大而增大：
- q = 0：0分/轮
- q = 1：3分/轮

也就是说：群体最优是全体合作，每人每轮 3 分，总计 3000 分。但个体激励却总鼓励偏离到 D，从合作生态中套利。因此该问题本质是一个社会困境。

## 3. 对其他囚徒策略的应对机制

因为看不到身份，不能做"对这个人报复"。只能做群体层面应对。

### 对"永远合作型"囚徒
这类囚徒总是 C。对他们单轮：
- 你选 D：得 5
- 你选 C：得 3

局部最优是 D。但若群体大量是永远合作者，你持续 D 会促使更多策略型囚徒转向 D，破坏生态。因此 PBAS 在高 p̂_t 时保留高合作概率 0.8，而不是 0。

**应对机制**：若观测到高合作率持续稳定，进入高合作模式；仍保留 20% D，避免完全被固定搭便车者压制。

### 对"永远背叛型"囚徒
这类囚徒总是 D。对他们：
- 你选 C：-2
- 你选 D：0

显然必须 D。

**应对机制**：若观测到 p̂_t 低于阈值 0.4，立即进入全 D 防御态；不再浪费合作试探，除非长期统计出现反转。

### 对"条件合作者"
例如：若最近观察到合作多，则倾向 C；若最近被背叛多，则转向 D。

这是最关键的群体，因为长期高收益生态要靠他们维持。

**应对机制**：在中高 p̂_t 区间提供足够合作信号；避免过早完全转 D，防止把条件合作者推入防御均衡；用少量 D 控制剥削风险。

### 对"随机策略"囚徒
假设对手以固定概率 r 合作。则单轮最优仍是 D，但其存在会拉低总体合作统计。

**应对机制**：将其视为噪声源；使用贝叶斯平滑估计而不是短窗平均，避免被短期波动误导；不因少量随机坏样本立刻坍塌为全 D。

### 对"伪装合作者后期收割型"
例如前期大量合作建立高 p，后期转 D 套利。在有限轮博弈里这是典型末期背叛问题。

**应对机制**：在接近末期（例如最后100轮）逐步降低合作概率：
```
q_t_final = q_t × (1 - (t-900)/100), t ∈ [900,1000]
```
即终局衰减合作，防止被尾盘收割。

## 4. 完整策略伪代码

```python
Initialize:
  nC = 0
  nD = 0
  alpha0 = 1
  beta0 = 1

For t = 1 to 1000:
  p_hat = (nC + alpha0) / (nC + nD + alpha0 + beta0)
  
  if t <= 50:
      cooperate with probability 0.2
  else:
      if p_hat < 0.4:
          cooperate with probability 0
      else if p_hat < 0.7:
          cooperate with probability ((p_hat - 0.4) / 0.3) * 0.5
      else:
          cooperate with probability 0.8
  
  if t >= 900:
      q = q * (1 - (t - 900)/100)
  
  Observe opponent action:
      if opponent played C: nC += 1
      else: nD += 1
```

## 5. 严格结论

**层1：严格个体最优**
- Always D
- 因为 D 严格支配 C
- 单轮与总期望都更高

**层2：若考虑群体反馈与长期生态稳定**
- PBAS 自适应混合策略
- 利用总体合作率估计
- 在高合作生态中参与合作，在低合作生态中防御
- 比"盲目合作"稳健，比"盲目背叛"更有可能在对称策略群体中维持高长期收益环境

## 最终结论

如果按严格博弈论、以"我的期望总分最大化"为唯一目标，最优策略是：**始终背叛**。

如果要求设计一个在大规模随机匹配、有限信息、群体反馈下更稳健的长期策略，则采用基于总体合作率估计的**自适应混合策略**更有解释力。
